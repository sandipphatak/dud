# ALightBox
ALightBox is an easy-to-use jQuery-Plugin which provides a lightbox for your website.

## Features
- Supports 9 different languages
	- German
	- English
	- French
	- Spanish
	- Italian
	- Chinese
	- Russian
	- Danish
	- Dutch
- Supports images
- Supports videos from YouTube
